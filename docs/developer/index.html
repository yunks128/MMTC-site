# Developer Documentation

Welcome to the Developer Documentation for MMTC! This section is designed to help developers get started with contributing to, adapting, or testing our software. Whether you're new to the project or looking to dive deeper into the codebase, you'll find all the resources you need right here.

## Introduction

MMTC (Multi-Modal Transport Control) is designed to facilitate efficient management and control of transportation systems across various modes. This developer documentation will guide you through everything you need to know to start contributing to the project, customizing the software, or running your own tests.

## Getting Started with Development

The [Getting Started with Development](getting-started-dev) guide provides an overview of the tools and practices you'll need to begin working on MMTC. It includes setup instructions, development workflows, and best practices tailored to our codebase.

- Initial setup and configuration for the MMTC environment
- Development workflows and processes unique to our project
