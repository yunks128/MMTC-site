# Frequently Asked Questions (FAQs)

Welcome to the Frequently Asked Questions (FAQs) section! Here you'll find answers to common questions about MMTC. Whether you're troubleshooting an issue or just curious about the software, this page is designed to help you find the information you need quickly.

## General Questions

### 1. What is MMTC?

**Answer**:  
MMTC (Multi-Modal Transport Controller) is a software solution designed to facilitate the management and optimization of transport operations across various modes of transportation. It is aimed at logistics companies, transport operators, and supply chain managers, providing tools to enhance efficiency and reduce costs.

### 2. How do I install MMTC?

**Answer**:  
You can install MMTC by following the instructions in the [Installation Guide](user/installation.md). The guide provides step-by-step instructions for different operating systems, including Windows, macOS, and Linux, as well as various installation methods.

### 3. What are the system requirements?

**Answer**:  
MMTC requires the following minimum system specifications:
- **Operating System**: Windows 10 or later, macOS 10.15 or later, Ubuntu 20.04 or later
- **Hardware**: Minimum 8GB RAM, 2GHz dual-core processor
- **Dependencies**: Java 11, PostgreSQL 12, Docker (for containerized deployments)

For more details, refer to the [Installation Guide](user/installation.md).
